# 快点装机

一个装机导航网站，提供装机模拟、系统下载、软件下载等一站式服务。

## 功能特性

- **装机模拟器**：智能推荐硬件配置，实时价格跟踪，兼容性检查
- **系统下载**：Windows、Linux、macOS等操作系统官方镜像下载
- **软件下载**：装机必备软件合集，包含驱动、浏览器、办公软件等
- **硬件指南**：详细的硬件选购指南和配置建议

## 技术栈

- **前端框架**：React 18
- **样式框架**：Tailwind CSS
- **路由管理**：React Router v6
- **构建工具**：Vite
- **数据存储**：JSON文件（无需后端）

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```
